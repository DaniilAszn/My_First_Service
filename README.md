1) C:\..\TimeSeriesAnalysis -> pip install -r requirements.txt
2) Terminal 1: cd ..\TimeSeriesAnalysis\backend -> uvicorn main:app --reload
3) Terminal 2: cd ..\TimeSeriesAnalysis\frontend -> streamlit run app.py
