# My_First_Service
Mentor's seminar. Pet-project with backend and frontend.

# Концептуальная структура проекта
Данный проект представляет из себя небольшой сервис для анализа временных рядов на примере данных yfinance. Пользователь выбирает для себя компанию из MAANG-группы, срез по временному интервалу для ряда, после чего ему потребуется нажать на одну кнопку "RUN ANALYSIS" и тогда он сможет увидеть подробную аналитику по выбранным им данным.

Анализ данных по шагам:
1) Загрузка данных из yfinance, преобразование исходного ряда и его демонстрация
2) Аналитика временного ряда: сглаживание, декомпозиция и проведение тестов на автокорреляцию и стационарность
3) Разработка модели для прогнозирования временного ряда и демонстрация прогнозной способности модели

# Как запустить сервис (локально)
1) C:\\<your_path_to_project>\TimeSeriesAnalysis -> pip install -r requirements.txt
2) Terminal 1: cd ..\TimeSeriesAnalysis\backend -> uvicorn main:app --reload
3) Terminal 2: cd ..\TimeSeriesAnalysis\frontend -> streamlit run app.py
